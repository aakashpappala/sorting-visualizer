# Sorting Algorithm Visualizer

## 📌 Overview

Sorting Algorithm Visualizer is a web-based application that demonstrates how different sorting algorithms work using interactive animations. It helps users understand the step-by-step execution of sorting techniques in a visual and intuitive way.

---

## 🚀 Features

* Visual representation of sorting algorithms using animated bars
* Supports multiple algorithms:

  * Bubble Sort
  * Selection Sort
  * Insertion Sort
  * Merge Sort
  * Quick Sort
    
* Adjustable array size
* Speed control for visualization
* Real-time animation of comparisons and swaps
* Displays time complexity for each algorithm

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)

---

## ▶️ How to Run

1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in any browser
4. Select:

   * Algorithm
   * Array Size
   * Speed
5. Click on **Sort** to visualize

---

## 🧠 How It Works

* Random array values are generated and displayed as vertical bars
* Selected sorting algorithm runs step-by-step
* Bars change height and color to represent:

  * Comparisons
  * Swaps
  * Sorted elements
* Animation delay is controlled by selected speed

---

## 🎯 Purpose

This project is designed to help beginners understand sorting algorithms through visual learning instead of only theoretical concepts.
